##Multiblog


###A TYPO3 Blog Extension


**New Features in this Version:**
* one Blog entry contains several content parts
* Ajax / EID for Comments
* n:m Category Relations
* SEO Fields for Blog and Posts

Testet in TYPO3 6.2 !
